📟 Your Commands

>mytttasks View your currently claimed tasks
  • Lists every task you have reserved with its ID and age
  • Use this to find task IDs before completing or unclaiming
>claimtttask <id> Claim an available task from the board above
  • Reserves the task so others can't claim it at the same time
  • Complete it with >completetask <id> to earn the points
>unclaim <id> Drop a task you claimed — puts it back as available
  • Only works on tasks you personally claimed
